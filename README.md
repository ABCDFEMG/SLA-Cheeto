## Usage

### How to Run
1. Inject the DLL to the game with your preferred injector.
2. Press insert to show the menu

## Features
- No Cooldown
- God Mode
- Damage Hack (Works for certain weapons only)
- Dumb Enemies (They don't attack you)
- Mission Time 
- Time Scale
- FPS Unlocker
- Fov Changer
- Skip Intro Movie
- Disable WebView
